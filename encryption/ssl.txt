1. Generate private key : openssl genrsa -out domain.key 2048
2. Denerate a Cert signing request : openssl req -new -key domain.key -out domain.csr
3. Generate private key and CSR in one command :
    openssl req -new -newkey rsa:2048 -keyout domain.key -out domain.csr
4. Checking CSR file : openssl req -text -in domain.csr -noout
